# LangChain Learning

Personal learning repo to understand the LangChain/LangGraph/DeepAgents ecosystem through progressive hands-on exercises.

See [PLAN.md](./PLAN.md) for the detailed learning plan and TODOs.
See [docs/00-ecosystem.md](./docs/00-ecosystem.md) for the ecosystem glossary.

## Quick start

```bash
npm install
cp ../evaneos/di-agent-ui/.env .env
cp ../evaneos/di-agent-ui/.env.local .env.local

# Run an exercise
npx tsx exercises/01-hello-llm/index.ts

# Run the Next.js app (exercises 10+)
npm run dev
```
